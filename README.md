# Calistenia Militar - Tu Plan 21 Días

![Tactical Dashboard Mockup](https://raw.githubusercontent.com/username/repo/main/public/og-image.png)

Una aplicación web de calistenia diseñada con una estética táctica y militar "Premium SaaS". Este proyecto transforma un plan de entrenamiento complejo de 21 días en una experiencia de usuario fluida, compacta y responsiva.

## 🚀 Características

- **Diseño Ultra-Compacto**: Nombres de ejercicios y números integrados para maximizar el espacio en pantalla.
- **Navegación sin Modales**: Procedimientos técnicos desplegables (accordions) para ver instrucciones sin perder el contexto de la lista.
- **Briefing de Misión**: Objetivos diarios y descripciones técnicas para cada jornada.
- **Crono de Recuperación**: Temporizador integrado para controlar los tiempos de descanso entre series.
- **Plan Nutricional Dinámico**: Guía de alimentación adaptada al tipo de entrenamiento (Fuerza, HIIT, Mixto).
- **Control de Progreso**: Registro de misiones cumplidas con barra de progreso interactiva.
- **Video Enlaces**: Acceso directo a demostraciones en video (Google Drive) para técnica perfecta.

## 🛠️ Tecnologías

- **React + Vite**: Para un rendimiento ultra-rápido.
- **Tailwind CSS**: Estilizado moderno y responsivo.
- **Framer Motion**: Animaciones fluidas y transiciones suaves.
- **Lucide React**: Iconografía táctica.

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/PlanCalisteniaMilitar.git
   ```
2. Instala dependencias:
   ```bash
   npm install
   ```
3. Inicia en desarrollo:
   ```bash
   npm run dev
   ```
